# Kirsty & Jet 💙

A sweet, interactive couple's website with glassmorphism design, theme switching, and more.

## Features

- 🎨 6 color themes (Sky Blue, Rose Pink, Lavender, Mint Green, Sunset, Cherry Red)
- 🌸 Shared interests section with hover effects
- 🪣 Interactive bucket list (add, edit, delete, mark as done)
- 📅 Calendar with event tracking
- 🎵 Background music player
- ✨ Cursor sparkle effects
- 💙 Floating emoji animations
- 📱 Fully responsive design
- 💾 All data saved to localStorage

## Project Structure

```
├── assets/

```

## Setup

1. Drop your music file into `assets/` and rename it to `music.mp3`
2. Open `index.html` in a browser — or host on GitHub Pages

## Deploy to GitHub Pages

1. Create a new repo on GitHub
2. Push this folder:
   ```bash
   cd love-site
   git init
   git add .
   git commit -m "our little corner of the internet 💙"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
   git push -u origin main
   ```
3. Go to repo **Settings → Pages → Source → main branch → / (root)**
4. Your site will be live at `https://YOUR_USERNAME.github.io/YOUR_REPO/`
